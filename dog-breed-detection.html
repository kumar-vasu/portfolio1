Dog BreedMulti class classification

Using transfer learning

In this project we worked on multi-class classification of dog breeds, we used the Dog Breed Identificationdataset from kaggle consisting of 10,000+ labelled images of 120 dog breeds. We achieved a 0.7 accuracyon the val and test sets. We achieved this by using mobilenet v2 model achitecture.

The optimizer used was ADAM and accuracy was used as the metric for evaluation.
