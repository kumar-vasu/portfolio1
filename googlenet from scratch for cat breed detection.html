Cat BreedMulti class classification

googlenet from scratch

In this project we worked on multi-class classification of cat breeds, we used the Cat Breed Identification dataset from kaggle consisting of 15,000+ labelled images of cat breeds. We achieved a 0.7 accuracy on the val and test sets. We achieved this by implementing googlenet achitecture from scratch. The optimizer used was ADAM and accuracy was used as the metric for evaluation.
