Face Recognition1:k matching problem

Triplet loss function

In this project we had to perform face recognition (references are linked below). Face recognition is a 1:k matching problem that answers the question "who is this person?" as compared to a 1:1 matching problem of face verification which answers the question if this person is the same person he is claiming to be. We used Facenet architecture for this purpose. FaceNet learns a neural network that encodes a face image into a vector of 128 numbers. By comparing two such vectors, you can then determine if two pictures are of the same person. We also applied the triplet loss function. Triplet loss is an effective loss function for training a neural network to learn an encoding of a face image.

Florian Schroff, Dmitry Kalenichenko, James Philbin (2015). FaceNet: A Unified Embedding for Face Recognition and Clustering

Yaniv Taigman, Ming Yang, Marc'Aurelio Ranzato, Lior Wolf (2014).DeepFace: Closing the gap to human-level performance in face verification
