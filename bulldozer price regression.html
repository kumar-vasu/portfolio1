Bulldozer price estimation

Regession problem

In this project we predict the auction sale price for a piece of heavy equipment to create a "blue book" for bulldozers.We used Blue Book for Bulldozers data set from kaggle. The data had total 53 columns. This is a regression problem with time series component so we didnot use cross validation. We perform EDA and align data according to sales date. We performed regression using random forest regression. Tuned hyperparamters using random search and group search cv. Performed evaluation using RMSLE, MAE, R^2. Resulting in Training MAE: 2927.55, Valid MAE: 5925.13, Training RMSLE: 0.14, Valid RMSLE': 0.24, Training R^2: 0.95, Valid R^2: 0.88.
