Volume controlwith hand gesture

mediapipe and open cv

This is a hobby project, It uses mediapipe and opencv to create a volume controller using real time image recognition. This works on a custom hand tracking module that uses mediapipe hand tracking to detect and track points on human hand in real time using only cpu. The module is made capable of returning the number of fingers that are raised, which fingers are raised and also there respective distance. We regulate the volume of device by increasing or decreasing the distance between fingers.
