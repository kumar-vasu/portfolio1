Heart diseaseclassification

Random Forest Classification

In this project we worked on heart disease classification, predicting the probability of a patient to ecounter heart disease. The original data came from the Cleveland database from UCI Machine Learning Repository.Howevever, we've downloaded it in a formatted way from Kaggle. The original database contains 76 attributes, but here only 14 attributes will be used. We performed EDA on the data and found visible markers increasing the probability of heart disease. We tried KNN, Random Forest and logistic regression as model choices with 0.688, 0.83 and 0.88 correct answer probability respectively. We performed hyperparameter tuning using random search and grid search. We perfomed evaluation using precision, recall, f1 score, classification report, ROC Curve and AUC.
